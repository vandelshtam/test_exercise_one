## How to install
- clone this project
- install vendor laravel
```bash
composer install
```
- make env
```bash
cp .env.example .env
```
- generate key
```bash
php artisan key:generate
```
- run migrations
```bash
php artisan migrate --seed
```
- install vendor javascript
```bash
npm install
```
- build manifest
```bash
npm run build
```
## Run project
- start server
```bash
php artisan server
```
